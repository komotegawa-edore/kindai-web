# 最初の50冊を売り切る計画

## 前提

- 出版審査: 1週間以内に通過見込み
- 書籍: 近大英語 長文読解ドリル（Kindle ¥780 / ペーパーバック ¥1,580）
- 資産: Webサイト（LP + 模試 + ブログ）、X 450フォロワー（受験生向け・休眠中）、塾の知り合い
- 目標: 出版後2ヶ月で50冊

## 販売チャネル別の目標内訳

| チャネル | 目標 | 難易度 |
|----------|------|--------|
| X（直接告知） | 10冊 | 低 |
| 塾・予備校への紹介 | 10冊 | 低 |
| Web模試 → 書籍CTA | 10冊 | 中 |
| Amazon検索（自然流入） | 10冊 | 中 |
| ブログSEO → Amazon | 10冊 | 中〜高 |
| **合計** | **50冊** | |

## Amazonレビュー獲得戦略

レビューは購入の意思決定に直結する。最初の5件が重要。

- 塾の講師に献本 → レビュー依頼（本音で書いてもらう）
- 模試受験者にメールで感想を聞く（将来的にはアカウント登録者へ）
- X で「購入した方、感想をAmazonレビューに書いていただけると嬉しいです」と投稿
- ※ やらせレビュー・対価を渡してのレビューはAmazon規約違反なので絶対にNG

## まとめ: 50冊販売の方程式

```
X告知（初速10冊）
  + 塾への紹介（10冊）
  + 模試 → 書籍CTA（10冊）
  + Amazon検索流入（10冊）
  + ブログSEO流入（10冊）
  = 50冊
```

どのチャネルも単独で50冊は難しいが、5つのチャネルを並行運用すれば十分達成可能。最初の1週間で10〜15冊売ることが、Amazon検索順位を上げて残りの流れを作る鍵。
