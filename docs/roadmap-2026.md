# ドリレオ ロードマップ 2026 — 年間1000冊販売計画

## 目標

大学別特化型の問題集を横展開し、2026年度中に累計1000冊の販売を達成する。

## 目標の分解

| 時期 | 月間目標 | 書籍ラインナップ |
|------|----------|------------------|
| 4〜6月 | 月20冊 | 近大英語 1冊 |
| 7〜9月 | 月60冊 | 近大 2〜3冊 + 関大 1冊 |
| 10〜12月 | 月150冊 | 4〜6大学 × 各1〜2冊 |
| 1〜3月 | 月200冊 | 受験直前期の追い込み需要 |

書籍の「種類」を増やすことが売上に直結する。

## マネタイズ戦略

### 収益の柱

| 収益源 | 優先度 | 備考 |
|--------|--------|------|
| 書籍販売（印税） | 最優先 | Kindle + ペーパーバック |
| Amazonアソシエイト | 高 | 他社参考書の紹介料3% |
| 有料模試パック | 中 | 無料模試ユーザーが増えてから |
| サブスク（月額制） | 低 | ユーザー基盤ができてから |

### 集客チャネル

| チャネル | 施策 |
|----------|------|
| SEO（ブログ） | 大学別の対策記事でロングテールキーワード獲得 |
| SEO（LP） | 大学別LPで「○○大学 英語 対策」を狙う |
| Web模試 | 無料模試 → 結果画面で書籍CTA |
| Amazon検索 | タイトル・説明文のキーワード最適化 |
| SNS | X / Instagram で受験生コミュニティにリーチ |
